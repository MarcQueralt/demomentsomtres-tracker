Converteix WordPress en un repositori per fer de tracker

== Description ==
Converteix WordPress en un repositori per a fer tracking d\'informació provinent d\'aplicacions diverses.
Incorpora llibreria per a cridar-ho remotament.

== Installation ==
* S\'instal·la d\'acord al procediment habitual
* Cal copiar els fitxers de la carpeta root a l'arrel de WordPress.

== Frequently Asked Questions ==
=== Adreça dels webservices ===
Els webservices es criden a través del fitxer dms3-tracker.php de l'arrel de WordPress.
=== Llibreria per a fer les crides ===
A la carpeta lib-client hi ha una llibreria per a mostrar els continguts.
=== Test ===
Es pot verificar el correcte funcionament del component invocant el fitxer test.php de la carpeta test.
